[h1]All eXpansion plugins[/h1]
[url=/info/adm]Adm[/url] - Visual server administration using gui windowing system 
[url=/info/admingroups]AdminGroups[/url] - a sophisticated permission-based system
[url=/info/autoload]AutoLoad[/url]- a support plugin to load all standard plugins easily
[url=/info/autotagexpansion]AutoTageXpansion[/url] - is a serverTags plugin to provide additional information about the server
[url=/info/autotrackmanager]AutoTrackManager[/url] - automatically remove maps with bad mapratings
[url=/info/chat]Chat[/url] - provides a customized chatbox style
[url=/info/chatadmin]Chat_Admin[/url] - admin commands in traditional chat-based way
[url=/info/chatlog]ChatLog [/url] - provides last 100 lines of chat history browsable in a window
[url=/info/checkpointcount]CheckpointCount[/url] - is a widget familiar from XAseco, simply displays current checkpoint at bottom of the screen
[url=/info/core]Core[/url] - the basis of the plugin pack
[url=/info/database]Database[/url] - creates and manages various database tables used with eXpansion
[url=/info/dedimania]Dedimania[/url] - obviously, a dedimania integration plugin!
[url=/info/docgen]DocGen[/url] - generates html documentation of all available the chat commands with help
[url=/info/donatepanel]DonatePanel[/url] - a donate panel, to allow planets donation made for the server 
[url=/info/emotes]Emotes[/url] - add various emotes like /gg
[url=/info/faq]Faq[/url] - a ingame help browser with many topics
[url=/info/forcemod]ForceMod[/url] - force a custom mod-graphics for the server
[url=/info/gui]Gui[/url] - support plugin
[url=/info/helpers]Helpers[/url] - support plugin
[url=/info/idlekick]IdleKick[/url] - idlekick
[url=/info/joinleavemessage]JoinLeaveMessage[/url] - Informs the server about joining and leaving players
[url=/info/localrecords]LocalRecords[/url] - provides local server records table
[url=/info/maniaexchange]ManiaExchange[/url] - ManiaExcange integration
[url=/info/mapratings]MapRatings[/url] - basicly a map karma plugin. This uses 5-star system to rate the map
[url=/info/maps]Maps[/url] - maps listing plugin, you can also sort the columns by clicking the column header, first time a-z, and second z-a
[url=/info/menu]Menu[/url] - a menu plugin for easy access to plugins so no chat commands are needed
[url=/info/mumble]Mumble[/url] - mumble server integration widget.
[url=/info/musicbox]MusicBox[/url] - add custom musics to be played in background music.
[url=/info/notifications]Nofications[/url] - a 2nd chat-like window with 6 scrolling lines to redirect different plugins server messages
[url=/info/personalmessages]PersonalMessages[/url] - adds personalmessages and admin group chat abilities
[url=/info/players]Players[/url] - a players list with kick, ban, blackslist, mute, force spec for admins
[url=/info/quiz]Quiz[/url] - run a quiz with points counting, powered by /ask command, no predefined questions data, players ask and answer theirself
[url=/info/tmkarma]TMKarma [/url] - TMKarma integration
[url=/info/widgetsbestcheckpoints]Widgets_BestCheckpoints[/url] - familiar widget to center top of the screen
[url=/info/widgetsbestruns]Widgets_BestRuns[/url] - a best runs widget to center top of the screen, choose wither this or the bestcheckpoints
[url=/info/widgetsclock]Widgets_Clock[/url] - adds server name and clock to top left corner of the screen
[url=/info/widgetsendrankings]Widgets_EndRankings[/url] - Shows a local server ranks at podium
[url=/info/widgetspersonalbest]Widgets_PersonalBest[/url] - Shows little widget with your statistics at bottom right of the screen
[url=/info/widgetsrecord]Widgets_Record[/url] - shows dedimania and localrecords on TAB-layer (by default) or as widget on normal layer at left edge of screen
[url=/info/widgetstimes]Widgets_Times[/url] - shows difference to your personal best or top1 time of the map, triggers on checkpoint
[url=/info/votes]Votes[/url] - a voting plugin to replace native restart and skip votes
