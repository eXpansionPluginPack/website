[h1]Mumble[/h1]
mumble server integration widget.

For more information to Setup this:
https://code.google.com/p/mumblereader/wiki/HowToUse
only the .json is needed. .xml is not included for the moment.
		
[h1]how to enable[/h1]
add following line to [code]<ManiaLive>\config\config.ini[/code]
[source]
manialive.plugins[] = 'eXpansion\Mumble'
[/source]

[h1]Configuration options[/h1]

[source]
;;url of mumble server like for ex: http://shotgunfun.de/mumble/1.json is 
ManiaLivePlugins\eXpansion\Mumble\Config.url = 'http://lynx.tmfserver.com/mumble-django/mumble/1.json'
[/source]
