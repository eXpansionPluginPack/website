[h1]Chat[/h1]
Customizes the chatbox text.

[h1]how to enable[/h1]
This plugin is Autoloaded.

[h1]Configuration options[/h1]
[h2]publicChatColor[/h2]
public chat color for server

[b]Var Name : [/b]publicChatColor (string)
[h3]Default Configuration[/h3]
[code]ManiaLivePlugins\eXpansion\Chat\publicChatColor[] = $ff0
[/code]

[h2]otherServerChatColor[/h2]
public chat color for relay server

[b]Var Name : [/b]otherServerChatColor (string)
[h3]Default Configuration[/h3]
[code]ManiaLivePlugins\eXpansion\Chat\otherServerChatColor[] = $0d0
[/code]
