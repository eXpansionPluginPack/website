[h1]Quiz[/h1]
TMKarma.com integration.

[h1]how to enable[/h1]
add following line to [code]<ManiaLive>\config\config.ini[/code]
[source]
manialive.plugins[] = 'eXpansion\TMKarma'
[/source]

[h1]Configuration options[/h1]
[h2]countryCode[/h2]
in case autodiscover fails, set default country code here (3 letters uppercase)

[b]Var Name : [/b]countryCode (string)
[h3]Default Configuration[/h3]
[code]ManiaLivePlugins\eXpansion\TMKarma\countryCode[] = 
[/code]
