[h1]Quiz[/h1]
TMKarma.com integration.

[h1]how to enable[/h1]
add following line to [code]<ManiaLive>\config\config.ini[/code]
[source]
manialive.plugins[] = 'eXpansion\TMKarma'
[/source]

[h1]Configuration options[/h1]

The plugin will try to autosense your servers country by using geoip service.
You can define own country code if the autosense doesn't work.

[source]
ManiaLivePlugins\eXpansion\TMKarma\Config.countryCode = "deu";
[/source]
