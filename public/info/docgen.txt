[h1]DocGen[/h1]
generates html documentation of all available the chat commands with help.
the documentation file will be created to [code]Manialive\libraries\ManiaLivePlugins\eXpansion\DocGen\Docs[/code]

[h1]how to enable[/h1]
add following line to [code]ManiaLive\config\config.ini[/code]
[source]manialive.plugins[] = 'eXpansion\DocGen'[/source]

[h1]Configuration options[/h1]

none.
