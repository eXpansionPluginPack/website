[h1]DonatePanel[/h1]
Display a donatepanel for planets donation.

[b]To enable planets donations on server[/b]
Please add your server validation code (usually 5 capital letters) to [code]<dedicated server>\UserData\Config\dedicated_cfg.txt[/code] 
and send a initial 100 planets to your server account using in-game mail. 

[h1]how to enable[/h1]
This plugin is Autoloaded.

[h1]Configuration options[/h1]
none.
