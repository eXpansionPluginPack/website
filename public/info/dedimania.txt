[h1]Dedimania[/h1]
Dedimania integration plugin.

[b]Note![/b]
this plugin will save files to [code]<dedicatedserver>\UserData\Replays\Dedimania\[/code]
if your server is highly visited and new records are being driven, the folder may build up to take diskspace quite much, since a replay-file takes space typically from 300kb to 500kb!

[h1]how to enable[/h1]
add following line to [code]ManiaLive\config\config.ini[/code]
[source]manialive.plugins[] = 'eXpansion\Dedimania'[/source]

[h1]Configuration options[/h1]
[h2]login[/h2]
Use the login of the dedicated server

[b]Var Name : [/b]login (string)
[h3]Default Configuration[/h3]
[code]ManiaLivePlugins\eXpansion\Dedimania\login[] = 
[/code]

[h2]code[/h2]
Visit <a href="http://dedimania.net/tm2stats/?do=register">http://dedimania.net/tm2stats/?do=register</a> to get code for your server.

[b]Var Name : [/b]code (string)
[h3]Default Configuration[/h3]
[code]ManiaLivePlugins\eXpansion\Dedimania\code[] = 
[/code]
