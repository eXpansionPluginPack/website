[h1]MapRatings[/h1]
Basicly a map karma plugin. This uses 5-star system to rate the map.
Displays widget at end of map by default. Chat triggers: +++,++,+,-,--,---
[img]/img/rate_widget.jpg[/img]

[h1]how to enable[/h1]
This plugin is Autoloaded.

[h1]Configuration options[/h1]
[h2]sendBeginMapNotices[/h2]
Sends chat message of  current rating at map start

[b]Var Name : [/b]sendBeginMapNotices (bool)
[h3]Default Configuration[/h3]
[code]ManiaLivePlugins\eXpansion\MapRatings\sendBeginMapNotices[] = false
[/code]

[h2]showPodiumWindow[/h2]
enable showing maprating window at podium

[b]Var Name : [/b]showPodiumWindow (bool)
[h3]Default Configuration[/h3]
[code]ManiaLivePlugins\eXpansion\MapRatings\showPodiumWindow[] = true
[/code]
